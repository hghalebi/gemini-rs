# Contributing to gemini-rs

Thank you for your interest in contributing! We welcome pull requests, bug reports, and feature suggestions.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Create a new branch** for your feature or fix (`git checkout -b my-new-feature`).
4.  **Commit your changes** (`git commit -am 'Add some feature'`).
5.  **Push to the branch** (`git push origin my-new-feature`).
6.  **Create a Pull Request**.

## Development

*   Ensure all tests pass: `cargo test`
*   Format your code: `cargo fmt`
*   Run linter: `cargo clippy`

## Reporting Bugs

Please open an issue on GitHub with:
*   A clear description of the bug.
*   Steps to reproduce.
*   The version of `gemini-rs` and `gemini` CLI you are using.
